### Week 4 Goals:

- Learn about and implement Amazon ElastiCache (Redis/Memcached)
- Research Amazon CloudFront and the CDN model
- Learn about Lambda@Edge and how to handle requests/responses at Edge
- Get acquainted with and implement AWS Lambda (Serverless)
- Monitor the system with Amazon CloudWatch and Grafana
- Manage and operate the system with AWS Systems Manager

### Week 4 Achievements:

- Understanding and implementing **Amazon ElastiCache**, knowing how to use caching to optimize application performance.
- Understanding the operating principles of **Amazon CloudFront (CDN)** and how to distribute content globally.
- Understanding how **Lambda@Edge** handles user-facing logic to reduce latency.
- Using **AWS Lambda** to build serverless applications, eliminating the need for server management.
- Monitoring and tracking the system using **Amazon CloudWatch**, visualizing metrics with **Grafana**.
- Managing and operating EC2 more effectively through **AWS Systems Manager** (Session Manager, Parameter Store, Automation).
