- Learn and deploy Amazon Lightsail
- Get familiar with the AWS Cloud9 development environment
- Study and configure EC2 Auto Scaling
- Learn NoSQL database concepts with Amazon DynamoDB

- Practice AWS CLI: create and manage S3, EC2, IAM, VPC, and DynamoDB resources

### Week 3 Outcomes:

- Successfully understood and deployed **Amazon Lightsail** for lightweight and rapid application hosting.
- Gained hands-on experience using **AWS Cloud9** as a cloud-based development environment.
- Understood the **EC2 Auto Scaling** mechanism and how to automatically scale resources based on workload.
- Acquired foundational knowledge of **Amazon DynamoDB** and NoSQL data modeling concepts.
- Learned how to manage **DNS and domain services with Amazon Route 53**, including routing and health checks.
- Practiced using the **AWS CLI** to create and manage core AWS services such as S3, EC2, IAM, VPC, and DynamoDB.
